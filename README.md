# India Unemployment Rate Analysis (COVID Phases)

## Objective
To analyze India’s unemployment rate trends using real World Bank data and
study the impact of COVID-19 by dividing the period into pre-COVID, COVID,
and post-COVID phases.

## Dataset
- Source: World Bank (Unemployment, % of labor force)
- Country: India
- Frequency: Annual

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Methodology
- Data cleaning and handling of real-world CSV issues
- Transformation of wide-format data to long format
- Time-series trend analysis
- Phase-wise mean unemployment comparison:
  - Pre-COVID (<2020)
  - COVID (2020–2022)
  - Post-COVID (>2022)

## Key Insights
- Unemployment trends fluctuate over time in India
- COVID-period unemployment appears lower due to reduced labor force participation
- Post-COVID period shows adjustment pressures in the labor market
